# Voltrac Energy - Corporate Web Portal

A professional, full-stack website redesign for **Voltrac Energy** (a next-generation electric vehicle motive battery manufacturing company), specialized in high-performance lithium traction battery packs for the electric mobility sector.

---

## 1. Key Highlights & Features
*   **Hero Banners Slider**: Dynamic Framer-Motion based homepage slider showcasing advanced EV batteries, robotic assembly lanes, and motive charging integration.
*   **Product Showcase Grid**: Searchable, filtered vehicle battery catalog leading to dedicated details views (features list, battery application grids, specifications comparison profiles).
*   **Automotive motive solutions pages**: Clean technical breakdowns of lithium-ion systems tailored across four major vehicular categories:
    *   *Two-Wheeler Solutions* (Premium traction packs for e-scooters)
    *   *Three-Wheeler Solutions* (Heavy duty LFP packs for e-rickshaws)
    *   *Golf Cart Solutions* (Direct drop-in golf & resort transit batteries)
    *   *MHE & Forklift Solutions* (High-amp LFP forklift traction packs)
*   **Interactive Battery Specification Finder**: Side-by-side technical evaluation table highlighting nominal voltage, capacities, chemistries, and cycle life with custom visual parameter indicators.
*   **Safety Testing & Production Video Gallery**: Direct loop player showcasing cell matrix robotic welding, robotic sorting, and rapid charging thermal validation.
*   **B2B Partnership Portal**: Dedicated online application logging for regional distributors, OEM aggregators, and fleet operations managers.
*   **Full-Stack Validation**: State-bound validations on all B2B and support ticket registration portals.

---

## 2. Technical Stack

*   **Frontend**: React (Vite-powered), React Router DOM (v6), Tailwind CSS, Framer Motion, Axios.
*   **Backend**: Node.js, Express.js.
*   **Data Storage**: Static JSON files.

---

## 3. Getting Started

### 3.1 Prerequisite Installations
*   Ensure [Node.js](https://nodejs.org/) (version 16+) is configured on your system.

### 3.2 Setting Up the Backend
1. Open a terminal in the `./backend` folder:
   ```bash
   cd backend
   npm install
   ```
2. Start the Express server:
   ```bash
   npm start
   ```
   *The mock server will listen at `http://localhost:5000`.*

### 3.3 Setting Up the Frontend
1. Open a separate terminal in the `./frontend` folder:
   ```bash
   cd frontend
   npm install
   ```
2. Start the Vite React development server:
   ```bash
   npm run dev
   ```
   *The React UI will run at `http://localhost:5173` (or the nearest available port).*
